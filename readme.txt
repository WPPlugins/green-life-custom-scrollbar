== Description ==

Modern and sexy scrollbar for desktop and mobile browsers from Green Life IT. You can customize the scrollbar as you want.

See Demo: http://greenlifeit.com/

You can change scrollbar options from settings menu. To customize your scrollbar go to Settings > GL Scrollbar.

== Installation ==

Install from wordpress.org:

01. Go to Plugins > Add New.
02. Search for "Green Life Custom Scrollbar"
03. Install and Active it.
04. To customize scrollbar go to Settings > GL Scrollbar.

Install from ZIP file:

01. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
02. Activate the plugin through the 'Plugins' screen in WordPress
03. Use Settings->GL Scrollbar screen to customize the plugin
